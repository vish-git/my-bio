This is a IOT application which gathers and stores sensor data using MQTT and MongoDB, apply business rules, expose data using rest api services for front end to consume. My role is IOT developer and my responsibility is to develop process flow using Node-Red to store and expose data using MQTT/MongoDB and to develop functions to apply business rules.This POC is developed using Javascript , Node , Raspberry PI 3 ,Node-Red Raspberry-pi device is used for processing.
